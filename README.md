# Fast Graph 

Fast Graph is a higher order framework, composed from the web framework technologies to build an ultra-fast, type-safe, and modern API from the Ground up on top of the Neo4j Graph Database.

## Technologies Used 

* *Web Servers*:  
    * [Uvicorn (default)](https://github.com/encode/uvicorn)
    * [Hypercorn](https://gitlab.com/pgjones/hypercorn/)
    * Use your Own (Link to docs incoming)

* GraphQL Server: [Strawberry](https://strawberry.rocks/docs)
* Rest Framework: [Fast API](https://fastapi.tiangolo.com/)
